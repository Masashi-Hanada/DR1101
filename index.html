DR-1101 使い方
▶ 基本操作
・START/STOP（またはスペースキー）で再生/停止
・TEMPOノブでBPM調整（ドラッグまたは±ボタン）
・STEPノブでステップ数（1〜16）変更
・SWINGノブでグルーブ感を追加
▶ ステップ入力
・LCDのマス目をタップ/クリックでON/OFF
・VELボタンONで 弱→中→強→OFF のサイクル入力（FLMと排他）
・FLMボタンONでフラムモード：ノートのあるマスをタップしてフラムON/OFF（右上に三角マーカー点滅）
・フラムONのステップは28ms前に小さなゴーストヒットが入る
・ノートを消すとフラムも同時にクリアされる
・ドラッグで連続入力可能
▶ パターン管理
・PAT 1〜16でパターン切り替え（再生中は1小節末に反映）
・BANK A〜Dでバンク切り替え（64パターン）
・編集した内容は WRITE を押すまで保存されません
・別パターンに移動すると編集内容は破棄されます
・WRITEボタンが点灯（ピンク）= 未保存の変更あり
・FILLボタンでFILLモード（パターン13〜16が1小節再生して戻る）、RANDでランダムパターン生成
・UNDOで直前の操作を取り消し
▶ MIX画面
・VOL/TUNE/DECAY/TONE/SAT/REVをパートごとに調整
・パート名長押し=MUTE、ダブルタップ=SOLO
・REVERBはSCH（軽量）/ IR（高品質）切り替え可
・FX2はマスターエフェクト（Isolator/LPF/HPF等）
【右端のPADボタン】
・タップ1回 → 単発再生
・タップ2回 → 4分音符でリピート再生（青が濃くなる）
・タップ3回 → 8分音符でリピート再生（紫がかった青）
・■ボタン → リピート停止
▶ Sound Set
・Sound Setは8パート分のサンプル＋MIX/FX設定をまとめて保存したセットです
【サンプルを登録する手順】
① SETTINGタブ → SAMPLE LOADエリアで各パートにWAVをロード
② MIX/FXを好みに調整
③ SAVE SOUND SET A〜D のいずれかを押して保存
【切り替え】
・フッターの Sound Set A〜E をタップで即時切り替え
・Sound Set E = 全パートを合成音にリセット（サンプルなし）
【他の人と共有する】
・EXPORT SET A〜D（緑ボタン）でサンプル込みのJSONファイルを書き出し
・受け取った人は IMPORT SET（青ボタン）でJSONを選択して読み込み
・データが空のスロットはボタンが薄く表示されます
▶ SONG MODE
・最大16スロットにパターンを並べて再生
・スロットごとにBANK/PTN/リピート/BPM/コメントを設定
・行をドラッグで並べ替え可能
▶ データ管理
・編集内容は自動保存（次回起動時に復元）
・EXPORT JSONで全データをファイル保存
・LOAD ALL PRESETで工場出荷状態に戻す
▶ LOCK / CLOCK OUT
・MIX/REVERB/FX1/FX2を個別ロックでパターン切替時に設定固定
・CLOCK OUTでオーディオパルス + MIDIクロックマスター出力
